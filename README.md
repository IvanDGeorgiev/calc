#Basic Calculator

This is a demo application

##How to use
### Supported Operations

"+, -, *, /, (,  )"

### Endpoint

calc?query=expression

where expression is a base64 encoded String

###Example:

####Original query: 
2 * (23/(3*3))- 23 * (2*3)

####With encoding: 
MiAqICgyMy8oMyozKSktIDIzICogKDIqMyk=
